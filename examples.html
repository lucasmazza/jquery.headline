Vanilla Usage

On your javascript:

$("element").headline(["One", "Two", "Three"]);

And don't forget about the css:

.heading-off {visibility: hidden;}

Fade in effect using CSS animations

Instead of the suggested .headline-off { visibility:hidden; } you can use this:

span{-webkit-transition:opacity 1s linear;}
.headline-off{opacity:0;}
.headline-off .cursor{visibility:hidden;}
.cursor{opacity: 1;}

Faster animations

$("element").headline({
  elements: ["One", "Two", "Three"],
  timeout: 10
});
